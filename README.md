# (C# 코딩) 에코 메신저 EchoMessenger
## 개요- C# 프로그래밍 학습- 1줄 소개: 사용자 키보드 입력을 받아서 처리하는 프로그램
## - 사용한 플랫폼: - C#, .NET Windows Forms, Visual Studio, GitHub
## - 사용한 컨트롤:- Label, TextBox, ListBox, Button
## - 사용한 기술과 구현한 기능 : Visual Studio를 이용하여 UI 디자인, Trim함수를 이용한 공백 제거, 수업중 가르쳐 주신 Time으로 시간값 구현, KeyDown 이벤트를 사용하여 Enter로도 전송이 가능하도록 구현, 출력창 인덱스 값으로 삭제 버튼 로직 예외 구현
